# BLOGs

A simple blog written in Express.js

## Frameworks

- node.js
- express.js
- mongoDB
- mongoose
- ejs
- bootstrap

## How to run

- `git clone https://github.com/Afnan-Navaz/BLOGs.git`
- `cd BlOGs`
- `npm i`
- set `MONGO_URI` env variable
- `npm start`
- go to /admin


[demo](https://bl0gs.herokuapp.com/blogs)
